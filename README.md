## 🎯 **What is This?**

This is a **fully interactive, dark-themed tracking website** designed to help you systematically prepare for Software Development Engineer roles over 12 months. It covers:

- ✅ **450+ LeetCode Problems** (organized by topic and difficulty)
- ✅ **Advanced Python** (decorators, generators, OOP, type hints)
- ✅ **Data Structures** (Arrays, Trees, Graphs, Heaps, etc.)
- ✅ **Algorithms** (Dynamic Programming, Greedy, Backtracking, Bit Manipulation)
- ✅ **System Design** (URL Shortener, Rate Limiter, LRU Cache, etc.)
- ✅ **Progress Tracking** (problems solved, streaks, achievements)
- ✅ **Local Storage** (your progress is saved automatically)

## 📖 **How to Use**

### **1. Navigation**
- Click tabs at the top to switch between phases:
  - **Overview**: Dashboard with stats and daily goals
  - **Phase 1 (M1-3)**: Python Mastery + DSA Foundation
  - **Phase 2 (M4-6)**: Advanced DSA (DP, Greedy, Backtracking)
  - **Phase 3 (M7-9)**: Mastery & Interview Prep
  - **Bonus (M10-12)**: Backend Frameworks & Distributed Systems
  - **Resources**: Books, courses, platforms, study tips

### **2. Track Progress**
- ✅ **Check boxes** as you complete topics/problems
- 📝 **Click "Update"** to change your current focus
- 🔄 Progress saves **automatically** to localStorage

### **3. Monitor Stats**
- **Top bar**: Overall progress, problems solved, streak
- **Dashboard cards**: Weekly summary, achievements
- **Phase cards**: Progress bars for each phase

### **4. Expand Details**
- Click on **week cards** to expand and see:
  - Daily schedule
  - Topics to master
  - LeetCode problems with difficulty tags
  - Python code snippets
  - Resources

### **5. Keyboard Shortcuts**
- `Ctrl/Cmd + S`: Manual save progress
- `Ctrl/Cmd + E`: Export progress as JSON

## ⚙️ **Customization**

### **Modify Problems**
Edit `data.js` to add/remove problems:
```javascript
problems: [
    { 
        id: "your-problem", 
        name: "Your Problem Name", 
        difficulty: "medium", 
        leetcodeNum: 123 
    }
]
```

### **Change Theme Colors**
Edit CSS variables in `styles.css`:
```css
:root {
    --accent-primary: #00ff88;  /* Change to your color */
    --accent-secondary: #00d4ff;
    --bg-primary: #0a0a0a;
}
```

### **Add Custom Phases**
Extend `data.js` with new phases/months following the existing structure.

## 🎓 **Learning Tips**

1. **Consistency is key**: 2-3 hours daily beats weekend marathons
2. **Understand, don't memorize**: Focus on patterns, not solutions
3. **Struggle before hints**: Spend 30-45 minutes before looking up solutions
4. **Review regularly**: Revisit problems after 1 day, 1 week, 1 month
5. **Explain out loud**: If you can't explain it, you don't understand it
6. **Track everything**: Use this tracker to monitor your progress
7. **Join communities**: LeetCode discussions, Discord servers, Reddit
8. **Mock interviews**: Practice with Pramp, interviewing.io

## 💡 **Pro Tips**

**For Maximum Results:**
1. Solve problems in order (don't skip around)
2. Write code by hand first, then type it
3. Analyze time/space complexity before submitting
4. Read discussions after solving (learn alternative approaches)
5. Maintain a separate notebook for patterns
6. Do weekly reviews of all problems solved
7. Participate in LeetCode contests (weekly practice)
8. Join study groups or find an accountability partner
